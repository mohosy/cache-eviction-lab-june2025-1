# Cache Eviction Lab

A CPU cache simulation playground for visualizing eviction policy behavior under the same memory access trace.

## Highlights
- Supports FIFO, LRU, and LFU policies.
- Adjustable cache size, address space, and trace length.
- Step-by-step or full-run simulation mode.
- Visual hit/miss timeline and live cache frame state.

## Stack
- HTML/CSS/Vanilla JavaScript
- Canvas timeline visualization

## Run
Open `index.html` in a browser.
